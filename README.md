# 🎮 坦克大战游戏 (Tank Battle Game)

一个使用 **Python** 和 **Pygame** 库从头开始构建的经典坦克大战游戏Demo。本项目完整实现了游戏的核心机制，包括双坦克对战、智能移动、物理碰撞、音效与动画，是面向对象编程和游戏逻辑设计的实践。

## ✨ 项目功能与特性

- **玩家坦克**：使用键盘方向键（↑↓←→）控制移动，空格键发射子弹。
- **敌方AI坦克**：
    - 具备自动随机移动和智能转向逻辑。
    - 拥有自主开火系统，以随机概率向玩家发射子弹。
- **完整的战斗系统**：
    - **子弹发射与碰撞**：敌我双方子弹独立计算，可实现相互击毁。
    - **实时的碰撞检测**：精确处理坦克与坦克、坦克与墙壁、子弹与所有元素的碰撞。
    - **爆炸动画效果**：击中目标后，会播放多帧序列爆炸动画。
- **地图元素**：
    - 可破坏的墙壁，拥有生命值设定，被多次击中后消失。
- **视听体验**：
    - 为子弹发射、击中、游戏开始等动作配备了音效。
    - 游戏状态实时显示，如敌方坦克剩余数量。

## 🛠 技术栈与实现亮点

- **编程语言**: Python 3
- **核心游戏库**: Pygame (用于处理图形、音效、事件和碰撞)
- **核心设计模式**: **面向对象编程 (OOP)**
    - 将游戏角色（坦克、子弹、墙壁、爆炸）抽象为独立的类，代码结构清晰，易于维护和扩展。
- **关键技术实现**:
    - **碰撞检测**: 使用 `pygame.sprite.collide_rect` 进行矩形碰撞检测，逻辑严谨。
    - **游戏状态管理**: 通过多个列表（如 `enemy_tank_list`, `bullet_list`）统一管理游戏内所有动态对象，并进行生命周期管理。
    - **事件驱动**: 采用 Pygame 事件循环，流畅处理键盘输入和游戏退出。
    - **资源管理**: 规范地加载和管理图片、音效等外部资源。

**🚀 如何运行游戏**

1.  **准备环境**：确保你的电脑已安装 Python 3。
2.  **安装依赖**：在终端中运行以下命令安装必需的库。
    ```bash
    pip install pygame
    ```
3.  **下载代码**：将本项目所有文件（包括 `img/` 文件夹和 `main.py`）下载到本地同一目录下。
4.  **运行游戏**：在终端中进入该目录，执行：
    ```bash
    python main.py
    ```
5.  **开始游戏**：
    - 使用 **方向键** 移动你的绿色坦克。
    - 使用 **空格键** 发射子弹。
    - 击败所有红色敌方坦克即可获胜。
